# Expert Level Bash Scripts - Blue Team Tools

## Overview

These scripts provide Blue Team detection and response capabilities for cryptocurrency miner infections. Use in authorized security testing and incident response only.

---

## Scripts

| ID | Name | Purpose |
|----|------|---------|
| EXP-B01 | Miner_Detection.sh | Detect running miners |
| EXP-B02 | Pool_Blocking.sh | Block mining pools |
| EXP-B03 | CPU_Monitoring.sh | Monitor for mining activity |
| EXP-B04 | Process_Hunter.sh | Hunt for hidden miners |
| EXP-B05 | Incident_Response.sh | Miner IR toolkit |

---

## Usage

All scripts should be run with appropriate permissions. Most require root/sudo for full functionality.

```bash
chmod +x EXP-B*.sh
sudo ./EXP-B01_Miner_Detection.sh
```
