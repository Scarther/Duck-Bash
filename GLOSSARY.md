# Glossary of Terms

## A

**AMSI (Antimalware Scan Interface)**
Windows interface that allows applications to integrate with antimalware products. Often targeted for bypass in attacks.

**APT (Advanced Persistent Threat)**
Sophisticated, long-term cyber attack typically conducted by nation-states or well-funded groups.

**ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge)**
MITRE's knowledge base of adversary behaviors based on real-world observations.

## B

**BadUSB**
Attack that exploits the reprogrammability of USB firmware to make devices appear as different device types (typically keyboards).

**Beacon**
Periodic communication from compromised system to command and control server.

**Blue Team**
Defensive security team responsible for detecting and responding to attacks.

## C

**C2 (Command and Control)**
Infrastructure used by attackers to communicate with and control compromised systems.

**Capslock/Numlock Exfiltration**
Technique using keyboard LED states to exfiltrate data from air-gapped systems.

**Credential Harvesting**
Collection of usernames and passwords from compromised systems.

## D

**Ducky Script**
Scripting language used for USB Rubber Ducky and similar devices to define keystroke injection payloads.

**Dwell Time**
Time between initial compromise and detection by defenders.

## E

**EDR (Endpoint Detection and Response)**
Security solution that monitors endpoints for suspicious activity and enables response.

**Enumeration**
Process of gathering information about a target system or network.

**Evil Twin**
Rogue access point that mimics a legitimate wireless network.

**Exfiltration**
Unauthorized transfer of data from a compromised system.

## F

**Firmware**
Low-level software that controls hardware devices.

**Flipper Zero**
Multi-function portable hacking tool with BadUSB capability.

## G

**GPO (Group Policy Object)**
Windows feature for centralized management of user and computer settings.

## H

**HID (Human Interface Device)**
USB device class for keyboards, mice, and other input devices.

**Handshake (4-Way)**
Authentication process in WPA/WPA2 that can be captured and cracked.

**Honeypot**
Decoy system designed to detect and analyze attacks.

## I

**IOC (Indicator of Compromise)**
Forensic artifact that indicates a security breach.

**IDS/IPS (Intrusion Detection/Prevention System)**
Security tools that monitor network traffic for suspicious activity.

## K

**KARMA Attack**
WiFi attack that responds to client probe requests for any SSID.

**Keylogging**
Recording keystrokes on a system.

**Kill Chain**
Stages of a cyber attack from reconnaissance to objective completion.

## L

**Lateral Movement**
Technique used to move through a network after initial access.

**LaunchAgent/LaunchDaemon**
macOS mechanisms for running persistent programs.

**LOLBins (Living Off the Land Binaries)**
Legitimate system tools used for malicious purposes.

## M

**MDM (Mobile Device Management)**
Enterprise solution for managing and securing mobile devices.

**MITRE ATT&CK**
Framework cataloging adversary tactics and techniques.

**MitM (Man-in-the-Middle)**
Attack where adversary intercepts communication between two parties.

## O

**OMA-URI**
Open Mobile Alliance Uniform Resource Identifier, used in MDM for device configuration.

**Obfuscation**
Techniques to make code or data difficult to understand.

## P

**Payload**
Code or commands executed as part of an attack.

**Persistence**
Techniques to maintain access to a compromised system across reboots.

**Phishing**
Social engineering attack using deceptive messages.

**PMKID**
Pairwise Master Key Identifier, can be used to crack WPA without full handshake.

**Privilege Escalation**
Gaining higher-level permissions on a compromised system.

## R

**Red Team**
Offensive security team that simulates real-world attacks.

**Registry (Windows)**
Hierarchical database storing configuration settings.

**Reverse Shell**
Shell session where the target connects back to the attacker.

**Rubber Ducky**
Commercial USB keystroke injection device by Hak5.

## S

**SIEM (Security Information and Event Management)**
Platform for collecting, analyzing, and correlating security logs.

**Sigma**
Generic signature format for SIEM detection rules.

**Social Engineering**
Manipulation techniques to trick people into security mistakes.

**Staged Payload**
Payload that downloads additional code from external source.

**Sysmon (System Monitor)**
Windows tool that logs detailed system activity.

## T

**TTP (Tactics, Techniques, and Procedures)**
Adversary behavior patterns used in threat intelligence.

**Tailgating**
Physical security attack where unauthorized person follows authorized person.

**Threat Hunting**
Proactive search for threats that evade existing controls.

## U

**UAC (User Account Control)**
Windows security feature requiring approval for privileged operations.

**USB Armory**
Open-source USB computer for security applications.

**USB Enumeration**
Process by which operating system identifies and configures USB devices.

## V

**VID/PID (Vendor ID/Product ID)**
Identifiers that specify USB device manufacturer and product.

**Volatility**
Memory forensics framework.

## W

**WIDS/WIPS (Wireless IDS/IPS)**
Security systems that monitor and protect wireless networks.

**WiFi Pineapple**
Commercial WiFi auditing device by Hak5.

**WPA/WPA2/WPA3**
WiFi Protected Access security protocols.

## Y

**YARA**
Pattern matching tool used for malware identification.

## Z

**Zero-Day**
Vulnerability unknown to the vendor or without available patch.
